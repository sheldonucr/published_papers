# Publications — Sheldon X.-D. Tan

**Browse online: https://sheldonucr.github.io/published_papers/**

Complete publication list and full-text PDF archive of **Prof. Sheldon X.-D. Tan**,
Department of Electrical and Computer Engineering, University of California, Riverside —
[VLSI System and Computation Lab (VSCLAB)](https://vsclab.ece.ucr.edu/).

## Browse online

A browseable version of the list is published with **GitHub Pages**:

> **https://sheldonucr.github.io/published_papers/**

Click any paper title to open its PDF. *(To enable the page: repository **Settings → Pages →
Build and deployment → Source: Deploy from a branch → `main` / root**.)*

## Statistics

- **359** reference entries across **Journal Articles**, **Commentary Journal Articles**, and **Symposia / Conference Proceedings**.
- **352** entries link directly to a full-text PDF in this repository (**364** PDF files total).
- **144** papers are catalogued with verified DOIs in `paper_doi_list.xlsx`.

The few unlinked entries are papers with no publicly retrievable PDF (older Chinese-language
journal articles, pre-DOI 1990s conference papers, an industry-consortium report, and a paper
not yet posted publicly).

## Usage

The PDFs are made available by the author for **personal and academic use**. Copyright for each
paper remains with its respective publisher (IEEE, ACM, Elsevier, Springer, MDPI, etc.); please
refer to the publisher's version of record (DOI) for citation.
